#### 通用制作间
------------------------------
>**适用产品**：杯子、台历、年历、挂历、鼠标垫、粘贴纸、LOMO卡、明信片、贺卡

    [URL]: http://zhigui.inguu.com/tool/bookmain/ + 产品ID  
    [例: 台历30020、杯子8727、LOMO卡30028、明信片30031]

--------------
>**制作流程**：
- 选择风格和模板 > 确认`[输入旺旺名]` > 上传照片 `[添加到模板、添加文字]` > 提交作品

--------------

##### 风格和模板选择  
* **风格选择**
    * 方式：左右滑动，点击选择
* **模板选择**
    * 选择风格后会显示相应的模板
    * 方式：上下滑动，点击选择
